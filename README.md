# nh_blog

A simple blog based on `python3.6` and `Django2.0`.

## requirement
- Django==2.0.3
- django-jet==1.0.7
- django-summernote==0.8.8.6
- pytz==2018.3
  


#### CN
- 文章管理，包括新增、删除和编辑
- 分类管理，包括新增、删除和编辑
- 标签管理，包括新增、删除和编辑
- 列表页展示概要信息、发布时间、分类、浏览次数
- 分页展示
- 点击"阅读全文"显示文章详细内容
- 详情页每刷新一次浏览次数+1
- 文章分类搜索
- 文章标签搜索
